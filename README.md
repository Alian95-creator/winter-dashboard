# ❄️ Winter Weather App

A modern **weather web application** built using **HTML, CSS, and Vanilla JavaScript**.
This project demonstrates **DOM manipulation, API integration, and interactive UI effects** such as loading animations and falling snow.

The application allows users to search for a city and view real-time weather data.

---

## 🌐 Live Preview

Click here :
https://alian95-creator.github.io/winter-dashboard/

---

## ✨ Features

* 🔎 Search weather by city name
* 🌡 Display temperature in Celsius
* ☁ Show weather condition and humidity
* ⏳ Loading spinner while fetching data
* ❄️ Falling snow animation for aesthetic winter theme
* 📱 Responsive layout

---

## 🛠 Technologies Used

* **HTML5**
* **CSS3**

* **DOM Manipulation**

## ⚙️ How to Run the Project


2. Open the project folder

3. Run `index.html` in your browser

---

## 🔑 API Setup

This project uses **OpenWeatherMap API**.

1. Sign up at
   https://openweathermap.org/api

2. Get your API key

3. Replace the API key in `script.js`

```
const API_KEY = "YOUR_API_KEY_HERE";
```
